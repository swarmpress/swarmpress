# swarm.press

swarm.press is a fully autonomous virtual publishing house operated by intelligent agents with human oversight.

## Features

- **11+ Autonomous Agents** - Specialized Claude agents (Writer, Editor, QA, Media, Linker, and more)
- **12 Temporal Workflows** - Content production, editorial review, publishing, batch processing, scheduling
- **Autonomous Scheduling** - Scheduled content publishing, media checks, link validation, stale content detection
- **Quality Assurance** - Automated content audits, link validation, media relevance checking
- **Event-Driven Architecture** - NATS JetStream with CloudEvents v1.0
- **GitHub Integration** - PRs for content review, Issues for tasks, OAuth authentication
- **Multi-Tenant** - Company → Department → Role → Agent hierarchy
- **Static Site Generation** - Astro-based website builder with 67 block types
- **Collections System** - Restaurants, accommodations, hikes, events, POIs, and more
- **Batch Processing** - Large-scale content generation and updates

## Technology Stack

| Layer | Technology |
|-------|-----------|
| Agent Runtime | Claude Agent SDK |
| Workflow Engine | Temporal.io + Schedules |
| Event Bus | NATS JetStream |
| Database | PostgreSQL |
| API | Express + tRPC (27 routers) |
| Admin Dashboard | Astro + React + shadcn/ui |
| Site Builder | Astro (67 block types) |
| Collaboration | GitHub (PRs, Issues, Webhooks, OAuth) |
| Media Storage | S3 / Cloudflare R2 |
