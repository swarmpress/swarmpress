# swarm.press

swarm.press is a fully autonomous virtual publishing house operated by intelligent agents with human oversight.

## Features

- **Autonomous Agents** - 4 Claude agents (Writer, Editor, Engineering, CEO Assistant)
- **Workflow Orchestration** - Temporal.io for durable, fault-tolerant workflows
- **Event-Driven** - NATS JetStream with CloudEvents v1.0
- **GitHub Integration** - PRs for content review, Issues for tasks
- **Multi-Tenant** - Company → Department → Role → Agent hierarchy
- **Static Site Generation** - Astro-based website builder
